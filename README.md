### What is this?

A simple python script to gain points on likestool.com

![Likestool](https://github.com/impshum/likestool/blob/master/screenshot.png?raw=true)

### Requirements

* Python3

### Instructions

* pip install BeautifulSoup, halo, splinter
* python run.py
* Run it

        python3 run.py

* Press Ctrl + C to exit.
